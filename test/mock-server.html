Docker不要 - Member拡張機能テスト用

⚠️ この方法について: このページはDockerが使えない場合の簡易テスト用です。
ブラウザの制約により、実際のSSEサーバーは動作しません。
Member拡張機能を直接呼び出す方法でテストします。
✅ 推奨される方法: 1. Docker Desktopをインストールして起動
2. docker-compose up -d でサーバーを起動
3. test/index.html でテスト

Dockerなしでテストする方法

1 Google Meetのページを開く
任意のGoogle MeetのURLを開きます（実際の会議でなくてもOK）
2 Member拡張機能をインストール
chrome://extensions/ で extensions/member フォルダを読み込む
3 このボタンをクリック
Google Meetのタブで直接オーバーレイUIを表示します
代替案: Python + Redisでローカル起動

# Redisをインストール（Windows用Memurai）
https://www.memurai.com/get-memurai

# Python環境で起動
cd server
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
